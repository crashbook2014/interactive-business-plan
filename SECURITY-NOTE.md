# Data hygiene statement

Everything in this repository is fictional demonstration data.

- All people are invented placeholders (Rania Kassem, Amal Rashed, Omar Hadi,
  Sara Nasser, Lina Fares, Karim Saleh, Noor Hamdan, Ziad Qassim,
  Maha Suleiman, Fadi Mansour, Huda Bakr). They do not correspond to real
  employees.
- All email addresses use the placeholder domain `pulse.sa` and are not
  deliverable mailboxes.
- Resident profiles, feedback, work orders, financial figures, corporate
  accounts (Meridian Energy, Northstar Consulting, Deema Tech, Crescent
  Partners) and property metrics are generated sample data, badged "sample"
  in the UI.
- The public demo runs entirely in the browser on this sample data; it stores
  nothing server-side and resets when the tab closes.
- No credentials, tokens, API keys, or secrets are committed anywhere in this
  repository. Production secrets are injected via environment variables /
  Secret Manager only (see DEPLOY.md).
